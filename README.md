# Usage

1. Clone this repo
2. Create `src/secret/BotData.ts` and fill it like this format:

```typescript
export class BotData {
  static EMAIL = '';
  static PASSWORD = '';
  static UUID = '';
  static NAME = '';
}
```

3. Run `src/index.ts`
